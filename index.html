Nodeflow Remote

Telegram Mini App

Control desktop runs, downloads and cleanup directly from Telegram. Every action is sent back to your Nodeflow desktop bot through Telegram WebApp data.

Use this for run and download. The value is stored locally in the Mini App.
How it works
  • Press an action button.
  • The Mini App sends JSON to your Telegram bot.
  • Nodeflow desktop receives it through long polling and executes the matching command.
  • The result comes back as a normal Telegram bot message.
